# Google Maps JavaScript Sample

## layer-traffic

Traffic layer sample.

## Setup

### Before starting run:

`npm i`

### Run an example on a local web server

`cd samples/layer-traffic`
`npm start`

### Build an individual example

`cd samples/layer-traffic`
`npm run build`

From 'samples':

`npm run build --workspace=layer-traffic/`

### Build all of the examples.

From 'samples':

`npm run build-all`

### Run lint to check for problems

`cd samples/layer-traffic`
`npx eslint index.ts`
