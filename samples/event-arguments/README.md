# Google Maps JavaScript Sample

## event-arguments

This sample shows how to access arguments from UI events.

## Setup

### Before starting run:

`npm i`

### Run an example on a local web server

`cd samples/event-arguments`
`npm start`

### Build an individual example

`cd samples/event-arguments`
`npm run build`

From 'samples':

`npm run build --workspace=event-arguments/`

### Build all of the examples.

From 'samples':

`npm run build-all`

### Run lint to check for problems

`cd samples/event-arguments`
`npx eslint index.ts`
