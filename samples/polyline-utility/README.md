## Setup

### Before starting run:

`npm i`

### Run an example on a local web server

`cd samples/polyline-utility`
`npm start`

### Build an individual example

`cd samples/polyline-utility`
`npm run build`

From 'samples':

`npm run build --workspace=polyline-utility/`

### Build all of the examples.

From 'samples':

`npm run build-all`

### Run lint to check for problems

`cd samples/polyline-utility`
`npx eslint index.ts`
