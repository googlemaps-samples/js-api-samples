# Google Maps JavaScript Sample

## event-click-latlng

This example listens for the click event, gets the latitude and longitude coordinates of the click from google.maps.MapMouseEvent.latLng, and displays those coordinates in an info window.

## Setup

### Before starting run:

`npm i`

### Run an example on a local web server

`cd samples/event-click-latlng`
`npm start`

### Build an individual example

`cd samples/event-click-latlng`
`npm run build`

From 'samples':

`npm run build --workspace=event-click-latlng/`

### Build all of the examples.

From 'samples':

`npm run build-all`

### Run lint to check for problems

`cd samples/event-click-latlng`
`npx eslint index.ts`
