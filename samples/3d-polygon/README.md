# Google Maps JavaScript Sample

## 3d-polygon

The 3d-polygon sample demonstrates how to create a 3D polygon.

Follow these instructions to set up and run 3d-polygon sample on your local computer.

## Setup

### Before starting run:

`$npm i`

### Run an example on a local web server

First `cd` to the folder for the sample to run, then:

`$npm start`

### Build an individual example

From `samples/`:

`$npm run build --workspace=3d-polygon/`

### Build all of the examples.

From `samples/`:
`$npm run build-all`
