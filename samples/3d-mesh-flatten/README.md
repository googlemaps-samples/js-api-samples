# Google Maps JavaScript Sample

## 3d-mesh-flatten

Sample generated from 3d-mesh-flatten

## Setup

### Before starting run:

`npm i`

### Run an example on a local web server

`cd samples/3d-mesh-flatten`
`npm start`

### Build an individual example

`cd samples/3d-mesh-flatten`
`npm run build`

From 'samples':

`npm run build --workspace=3d-mesh-flatten/`

### Build all of the examples.

From 'samples':

`npm run build-all`

### Run lint to check for problems

`cd samples/3d-mesh-flatten`
`npx eslint index.ts`
